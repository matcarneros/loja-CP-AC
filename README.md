#  Loja de Açaí e Cupuaçu

Um sistema simples de pedidos via terminal para uma loja que vende Açaí e Cupuaçu, com controle de sabores, tamanhos e cálculo do valor total do pedido.

---

## 🛠 Funcionalidades

- Exibição de cardápio com preços por tamanho e sabor
- Escolha de sabor (Açaí ou Cupuaçu)
- Escolha de tamanho (P, M ou G)
- Validação de entradas
- Cálculo automático do total do pedido
- Menu interativo em loop até o cliente encerrar

---

## 💻 Tecnologias Utilizadas

- Python 3
